Aide — comment ça fonctionne

            
Transcription (STT) + Traduction en direct

          

          
        

        

          
1) Deux moteurs, deux rôles

          

            STT (transcription) : transforme l’audio du micro en texte (offline via Vosk dans Electron).
            

            Traduction : transforme le texte reconnu en texte traduit (Google par défaut, ou Local si activé).
          


          
2) Réglages STT

          

            small (rapide) : dictée libre, meilleur choix pour conférences (vocabulaire ouvert).
            

            small + vocabulaire (précis) : utile si tu as une liste de termes/phrases attendus (réduit les “hallucinations”),
            mais peut brider la dictée si l’orateur sort du vocabulaire.
          


          
3) Import PPTX (option vocabulaire)

          

            Si tu as un deck, importe un .pptx pour extraire des termes (acronymes, noms propres, concepts) et remplir
            automatiquement le champ “vocabulaire/phrases attendus”.
          


          
4) Conseils pour de meilleurs résultats

          
    
            
  • Micro : le plus proche possible de l’orateur (distance/bruit = erreurs).
    • 
            
  • Silences : évite de laisser tourner sur une salle très bruyante si personne ne parle.
    • 
            
  • Conférences : préfère small (rapide). Utilise “vocabulaire” seulement pour termes clés.
    • 
            
  • Intermédiaire : le texte “intermédiaire” est moins fiable que le “final”.
    • 
          


          
5) Dépannage rapide

          

            Si la transcription paraît incohérente : baisse le bruit, rapproche le micro, ou reviens en small (rapide).
            Si un modèle met longtemps à se charger, regarde la barre de progression en haut.
          

        

      

    


    

      

        

          

            
Avant de commencer

            
Paramétrage rapide (micro → transcription → traduction)

          

          
        

        

          
1) Sélectionner le micro

          

            Dans Micro, choisis le micro à utiliser (idéalement un micro proche de l’orateur). Le vu‑mètre t’aide à
            vérifier que le bon micro capte bien la voix.
          


          
2) Choisir les langues

          

            - Langue parlée : la langue que tu entends (ou que tu parles).

            - Langue cible : la langue dans laquelle tu veux afficher la traduction.
          


          
3) Choisir les modèles

          

            - Modèle voix (Vosk) : la langue de transcription (offline en mode Electron).

            - Qualité STT :

              • small (rapide) : recommandé pour conférences (dictée libre).

              • small + vocabulaire (précis) : utile si tu as des termes (PPTX), réduit les erreurs hors‑sujet.
          


          
4) Démarrer / Arrêter

          

            Clique Démarrer pour écouter. Le texte reconnu apparaît dans “Texte reconnu”, puis la traduction dans “Traduction”.
            Clique Arrêter pour stopper l’écoute.
          


          
5) Fenêtre secondaire (popup)

          

            Clique Ouvrir / Fermer dans “Popup” pour afficher la fenêtre secondaire. Tu peux ensuite :
            choisir la taille, la police, le fond, activer Sans cadre, Premier plan et
            Redimensionnable, puis positionner la fenêtre où tu veux (ex: sur un écran secondaire).
          


          
6) Important — premiers chargements

          

            Au premier démarrage, certains modèles peuvent prendre du temps à se charger :
            

            - Vosk : initialisation du modèle (barre de progression en haut).

            - Traduction “Local” : téléchargement de llama-server et du modèle GGUF (peut être long).
            

            Une fois téléchargés, les démarrages suivants sont beaucoup plus rapides.
